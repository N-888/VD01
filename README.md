# VD01
HTML-таблица, адаптивно отображающаяся в браузерах на любых устройствах, которая показывает Рейтинг любимых / нелюбимых Блюд. Объемная Структура таблицы: 5 строк (рядов) Блюд и 4 столбца. Столбцы: Название блюда и краткое описание.  Оценка от 1 до 10 из 10 баллов.  Ссылка на рецепт.  Фото блюда. Изменяющийся цвет строк Таблицы при наведении.
